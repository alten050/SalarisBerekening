# Salaris Berekening

Kleine webapp om van een uurtarief naar een indicatieve salarisberekening te gaan.

## Starten

Open [index.html](/Users/richardaltena/Projects/Salaris%20berekening/index.html) direct in de browser of start lokaal een simpele server:

```bash
python3 -m http.server
```

Ga daarna naar `http://localhost:8000`.

## Opzet

- Gebaseerd op de drie voorbeeld-PDF's van Jan, Ming en Aiden
- Presets laden de voorbeeldwaarden en aannames
- Alle percentages zijn aanpasbaar, zodat je de tool per opdrachtgever of cao kunt bijstellen

## Belangrijk

De netto-uitkomst is indicatief. Voor een definitieve berekening blijven loonheffingstabellen, pensioenregeling en werkgeverslasten leidend.
